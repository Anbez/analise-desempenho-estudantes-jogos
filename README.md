# Análise de Impacto: Jogos vs. Desempenho Acadêmico

Este projeto analisa o impacto do hábito de jogar no desempenho acadêmico dos estudantes. A investigação busca entender se o volume de horas dedicadas aos jogos influencia diretamente as notas, a carga horária de estudo, a qualidade do sono e os níveis de vício em dispositivos de tela.

## 📊 Principais Insights

* A quantidade de horas jogando não possui forte influência nas horas de sono e de estudo dos indivíduos.
* **A Causa Raiz:** O vício em jogos/telas e o tempo de uso de dispositivos de forma geral apresentam forte correlação negativa com as notas, impactando diretamente o desempenho acadêmico.

## 🛠️ Tecnologias Utilizadas
* Python
* Pandas
* Seaborn
* Matplotlib
